# reticulum-pizero-monitor
Monitor the status of your Reticulum mesh network on the RaspberryPi Zero with ST7789 displays (Pimoroni)

## Features
- Real time system stats (CPU, RAM, Disk, Uptime, Architecture) 
- Reticulum daemon service monitoring (rnsd)
- LoRa RNode device detection (checks USB or ACM connections depending on radio)
- Multi screen interface with hardware button navigation (GPIO Pins = 5, 6, 16, 24)
- ST7789 240x240 display support

## Hardware Requirements
- Raspberry Pi Zero 2 WH (or compatible)
- Pimorani ST7789 240x240 SPI Display ( https://shop.pimoroni.com/search?q=pirate%20audio )
- LoRa RNode
